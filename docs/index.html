Application Android officielle

⬇ Télécharger l’APK

Après le téléchargement, si Android vous le demande, autorisez l’installation d’apps depuis « sources inconnues ».

Comment installer l’APK

  1. Appuyez sur « Télécharger l’APK » ci‑dessus.
  2. Une fois le fichier téléchargé, ouvrez‑le depuis la notification ou votre app Fichiers.
  3. Android affiche un avertissement. Choisissez « Paramètres » puis autorisez l’installation depuis cette source.
  4. Revenez en arrière et validez « Installer ».

Activer les « sources inconnues » par marque

1 Samsung (One UI)
  • Paramètres → Sécurité et confidentialité → Installer des applis inconnues.
  • Choisissez votre navigateur (Chrome, Samsung Internet, etc.) → Autoriser depuis cette source.
2 Xiaomi / Redmi / POCO (MIUI)
  • Paramètres → Mots de passe et sécurité → Confidentialité → Paramètres spéciaux.
  • Installer des applis inconnues → Sélectionnez le navigateur → Autoriser depuis cette source.
3 Huawei / Honor (EMUI / MagicUI)
  • Paramètres → Sécurité → Plus de paramètres → Installer des applis inconnues.
  • Choisissez l’app (navigateur / fichiers) → Autoriser cette source.
4 OPPO / realme (ColorOS)
  • Paramètres → Sécurité → Plus de paramètres → Installer des applis inconnues.
  • Sélectionnez l’app source → Autoriser depuis cette source.
5 OnePlus (OxygenOS)
  • Paramètres → Sécurité et confidentialité → Installation d’applis inconnues.
  • Choisissez le navigateur / fichiers → Autoriser.
6 Google Pixel (Android stock)
  • Paramètres → Sécurité et confidentialité → Plus de paramètres de sécurité.
  • Installer des applis inconnues → App source → Autoriser.
7 Motorola / Sony / Nokia / Nothing
  • Paramètres → Sécurité et confidentialité → Installer des applis inconnues.
  • Sélectionnez le navigateur ou l’app Fichiers → Autoriser.

Astuce: si l’option n’apparaît pas, recherchez « inconnues » dans la barre de recherche des Paramètres.

Questions fréquentes

Le bouton « Installer » n’apparaît pas

Éteignez le mode sombre ou activez l’orientation portrait, puis réessayez. Sur certains modèles, le bouton est masqué par une app de filtre d’écran: désactivez‑la le temps de l’installation.

L’APK est bloqué par Play Protect

Play Protect peut alerter pour toute app hors Play Store. Appuyez sur « Plus de détails » → « Installer quand même ».
